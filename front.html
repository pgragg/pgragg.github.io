Learning Code: A Roadmap

If you're just starting out and you want to learn the basics of web design, check out Codecademy's Web Development track. It will teach you

HTML, the content language of the internet, and

CSS, which you use to put all your content in the right place.

JavaScript is used to make web pages more interactive. Any time you see a button that pops up, a list that drops down, it was probably made with JavaScript. It is the most commonly required language of any developer.

Already know a little HTML and CSS?

You can start making websites right away! Just choose an online code editor from the tools section of the page, and start creating something right in your browser.

Need some inspiration? Jennifer Dewalt made 180 websites in 180 days.

Remember: you don't have to know how to make something perfect. Sites are never finished, after all, just abandoned.

Front End In Depth: Resources for Serious Study

Use the resources below if you have completed some of the above tutorials, or feel in need of an even bigger challenge.

The Eloquent Javascript book online is a huge free resource online. Don't worry about getting through the whole thing: if you're able to read through any of it, and remember even 10% of that, you've done yourself a huge favor.

Bootstrap is a name for standard prewritten CSS code that you can use to (more) easily make a great site. Twitter, Netflix, Paypal, and many other major companies built their sites using Bootstrap.
