Learning Code: A Roadmap

If you're just starting out and you want to learn the basics of web design, check out Codecademy's Web Development track.

JavaScript is used to make web pages more interactive. Any time you see a button that pops up, a list that drops down, it was probably made with JavaScript. It is the most commonly required language of any developer.

Bootstrap is a name for standard prewritten CSS code that you can use to (more) easily make a great site. Twitter, Netflix, Paypal, and many other major companies built their sites using Bootstrap.

Ruby is a fun programming language: try it out to see if you like it. If you like Ruby, and want to learn more about how to use your computer to solve problems, check out Codecademy's Ruby track.

Writing and Sharing Code Online

Write code that uses HTML, CSS, and JavaScript together! Use jsfiddle.net or codepen.io to create your project online. Feeling stuck? Look at other people's projects, fork them, and improve them!

Create your own free Github account, and become part of a huge online community of coders. Once you have a Github account you can post your projects online and improve upon other people's existing projects. Once you've coded your first webpage, you can put it on Github for free!
