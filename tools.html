Writing and Sharing Code Online

Once you know some code, you're going to want to test out some ideas! Maybe you have an idea for a JavaScript game, or want to test out making your own web page.

Write code that uses HTML, CSS, and JavaScript together! Use jsfiddle.net or codepen.io to create your project online. Feeling stuck? Look at other people's projects, fork them, and improve them!

Create your own free Github account, and become part of a huge online community of coders. Once you have a Github account you can post your projects online and improve upon other people's existing projects. Once you've coded your first webpage, you can put it on Github for free!

Habits of Great Programmers


Great programmers:

  • Have fun learning new skills. The fact that you joined this club says that you already have this fantastic quality.
  • Google everything that they want to know. They know that they can figure anything out if they keep Googling!
  • Know that every piece of knowledge is within their grasp.
  • Share their knowledge with others, and help their community.
  • Write short code that is easy to read.
  • Make comments on their own code.
